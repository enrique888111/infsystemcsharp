Лабораторная работа №4

Цель:

Научиться разрабатывать и перегружать методы с параметрами разного статуса и использовать встроенные методы.

Общие понятия

Методы – это основные строительные блоки классов, содержащие программный код. Структура метода выглядит следующим образом: [атрибуты] [спецификаторы] тип имя ([параметры]) { Тело метода; } Атрибуты предоставляют дополнительные указания компилятору относительно свойств метода, хотя их использование редко. Спецификаторы – это ключевые слова, определяющие доступность метода для других классов и указывающие тип. Имя метода – это идентификатор, используемый для его вызова, соблюдая правила именования переменных. Параметры – это список переменных, передаваемых в метод при его вызове. Тело метода содержит программный код, выполняющий определенную задачу. Если метод должен возвращать значение, используется ключевое слово return. Пример метода, вычисляющего выражение: public double Calc(double a, double b, double c) { if (a > b) return Math.Sin(a) * Math.Cos(b); else { double k = Math.Tan(a * b); return k * Math.Exp(c / k); } } Перегрузка методов позволяет создавать несколько методов с одинаковыми именами, но разными параметрами. Компилятор автоматически выбирает подходящий метод при компиляции программы. Пример перегруженных методов для возведения числа в степень: ///

/// Вычисление X в степени Y для целых чисел /// private int Pow(int X, int Y) { int b = 1; while (Y != 0) if (Y % 2 == 0) { Y /= 2; X *= X; } else { Y--; b *= X; } return b; } /// /// Вычисление X в степени Y для вещественных чисел /// private double Pow(double X, double Y) { if (X != 0) return Math.Exp(Y * Math.Log(Math.Abs(X))); else if (Y == 0) return 1; else return 0; } Вызов этих методов осуществляется одинаково, различие только в параметрах: Pow(3, 17); Pow(3.0, 17.0); Параметры по умолчанию позволяют задать значения для некоторых параметров метода, что позволяет опускать их при вызове. Пример метода с параметрами по умолчанию: private void GetData(int Number, int Optional = 5) { MessageBox.Show("Number: {0}", Number); MessageBox.Show("Optional: {0}", Optional); } Метод можно вызвать следующим образом: GetData(10, 20); GetData(10); Передача параметров по ссылке (ref) позволяет методу изменять значение переданной переменной. Пример метода с передачей параметра по ссылке: private void Calc(ref int Number) { Number = 10; } int n = 1; Calc(ref n); MessageBox.Show(n.ToString()); В этом случае на экране будет отображено число 10, так как значение переменной n было изменено в методе. Передача параметров по ссылке (out) также позволяет методу изменять значение переданной переменной, но не требует инициализации переменной перед вызовом метода. Пример метода с передачей параметра по ссылке (out): private void Calc(out int Number) { Number = 10; } int n; Calc(out n);
