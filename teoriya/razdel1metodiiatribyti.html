Раздел 1

Определение методов

Методы в C# определяют действия, которые могут выполняться объектами. Методы могут принимать параметры и возвращать значения.

Пример простого метода без параметров и возвращаемого значения:

public class HelloWorld
{
    public void SayHello()
    {
        Console.WriteLine("Hello, World!");
    }
}

Пример метода с параметрами и возвращаемым значением:

public class Calculator
{
    public int Add(int a, int b)
    {
        return a + b;
    }
}

Вызов метода:

Calculator calc = new Calculator();
int result = calc.Add(5, 3);
Console.WriteLine(result); // Output: 8

Методы могут быть перегружены, что позволяет создавать несколько методов с одним и тем же именем, но с различными параметрами.

public class Printer
{
    public void Print(string message)
    {
        Console.WriteLine(message);
    }

    public void Print(int number)
    {
        Console.WriteLine(number);
    }
}

Работа с атрибутами

Атрибуты в C# предоставляют метаданные для классов, методов и других элементов программы. Атрибуты используются для добавления дополнительной информации, которая может быть использована во время выполнения программы или на этапе компиляции.

Пример определения и использования атрибутов:

[System.AttributeUsage(System.AttributeTargets.Class | System.AttributeTargets.Method)]
public class AuthorAttribute : System.Attribute
{
    public string Name { get; }

    public AuthorAttribute(string name)
    {
        Name = name;
    }
}

[Author("John Doe")]
public class SampleClass
{
    [Author("Jane Doe")]
    public void SampleMethod()
    {
        Console.WriteLine("This is a sample method.");
    }
}

Чтение атрибутов с использованием рефлексии:

using System;
using System.Reflection;

public class Program
{
    public static void Main()
    {
        Type type = typeof(SampleClass);
        object[] classAttributes = type.GetCustomAttributes(typeof(AuthorAttribute), false);
        foreach (AuthorAttribute attr in classAttributes)
        {
            Console.WriteLine($"Class Author: {attr.Name}");
        }

        MethodInfo method = type.GetMethod("SampleMethod");
        object[] methodAttributes = method.GetCustomAttributes(typeof(AuthorAttribute), false);
        foreach (AuthorAttribute attr in methodAttributes)
        {
            Console.WriteLine($"Method Author: {attr.Name}");
        }
    }
}
