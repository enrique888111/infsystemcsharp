Принципы работы с XML-данными. Класс XmlDocument. Класс XPathNavigator

XML (eXtensible Markup Language) - это универсальный формат для обмена структурированными данными между различными приложениями. В C# для работы с XML-данными используются классы `XmlDocument` и `XPathNavigator`.

Класс `XmlDocument` позволяет загружать, создавать, изменять и сохранять XML-документы.

XmlDocument doc = new XmlDocument();
doc.Load("data.xml");

XmlNodeList nodes = doc.SelectNodes("/root/node");
foreach (XmlNode node in nodes)
{
    Console.WriteLine(node.InnerText);
}

Класс `XPathNavigator` предоставляет универсальный способ навигации и доступа к XML-данным с использованием языка запросов XPath.

XPathDocument doc = new XPathDocument("data.xml");
XPathNavigator nav = doc.CreateNavigator();

XPathNodeIterator nodes = nav.Select("/root/node");
while (nodes.MoveNext())
{
    Console.WriteLine(nodes.Current.Value);
}

Основы многопоточного программирования

Многопоточное программирование позволяет выполнять несколько потоков исполнения параллельно, что улучшает производительность и отзывчивость приложений.

В C# для работы с потоками исполнения используется класс `Thread`. Потоки позволяют выполнять различные части программы асинхронно.

Пример создания и запуска нового потока:

void ThreadMethod()
{
    // Логика потока
}

Thread thread = new Thread(ThreadMethod);
thread.Start();
