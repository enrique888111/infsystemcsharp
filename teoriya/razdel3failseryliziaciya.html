Раздел 3

Файлы. Виды файлов

В C# файлы могут быть использованы для хранения и обмена данными. Существует несколько видов файлов:

  • Текстовые файлы: хранят данные в текстовом формате и могут быть прочитаны и редактированы с помощью текстовых редакторов.
  • Бинарные файлы: содержат данные в двоичном формате и обычно используются для хранения структурированных данных, таких как изображения, звуковые файлы и т. д.
  • XML-файлы: представляют собой текстовые файлы, которые используются для хранения структурированных данных в формате XML.
  • JSON-файлы: также являются текстовыми файлами и используются для хранения данных в формате JSON.

Файлы и сериализация объектов

Сериализация в C# - это процесс преобразования объекта в поток байтов для его сохранения в файле или передачи по сети. Десериализация - обратный процесс, который восстанавливает объект из потока байтов.

Сериализация и десериализация могут использоваться для сохранения и загрузки объектов из файлов. В C# для этого часто используются классы BinaryFormatter для бинарной сериализации и XmlSerializer для сериализации в формате XML.

Пример сериализации объекта в бинарный файл:


using System;
using System.IO;
using System.Runtime.Serialization.Formatters.Binary;

[Serializable]
class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

class Program
{
    static void Main()
    {
        Person person = new Person { Name = "Alice", Age = 30 };

        BinaryFormatter formatter = new BinaryFormatter();
        using (FileStream fs = new FileStream("person.bin", FileMode.Create))
        {
            formatter.Serialize(fs, person);
        }
    }
}
