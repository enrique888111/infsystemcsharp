Раздел 3

Основы параллельного программирования

Параллельное программирование позволяет эффективно использовать мощности многоядерных процессоров для ускорения выполнения задач.

В C# для реализации параллельного программирования существует несколько подходов:

  • Многопоточность: Использование нескольких потоков для одновременного выполнения различных частей программы.
  • Асинхронное программирование: Использование асинхронных методов и операций для выполнения задач без блокировки основного потока.
  • Параллельные коллекции: Специальные коллекции данных, которые позволяют выполнять операции над элементами параллельно.
  • Параллельные запросы LINQ: Использование параллельной обработки данных при выполнении запросов с помощью LINQ (Language-Integrated Query).
  • Параллельные задачи и планировщик задач: Использование класса `Task` и планировщика задач для выполнения задач в параллельном режиме.

Пример использования параллельного выполнения с помощью `Parallel.ForEach`:

List<int> numbers = Enumerable.Range(1, 1000).ToList();
Parallel.ForEach(numbers, (number) =>
{
    Console.WriteLine($"Number: {number}, Thread ID: {Thread.CurrentThread.ManagedThreadId}");
});
