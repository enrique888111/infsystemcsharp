Раздел 4

Приложения с графическим пользовательским интерфейсом (GUI)

Создание приложений с графическим пользовательским интерфейсом (GUI) в C# обычно осуществляется с использованием Windows Forms или WPF (Windows Presentation Foundation).

Windows Forms предоставляет простой способ создания классических приложений с GUI, используя компоненты, такие как окна, кнопки, текстовые поля и т. д.

Пример создания простого приложения с использованием Windows Forms:

using System;
using System.Windows.Forms;

namespace MyFirstApp
{
    public class Program : Form
    {
        public Program()
        {
            Button button = new Button();
            button.Text = "Click me";
            button.Click += (sender, e) =>
            {
                MessageBox.Show("Button clicked!");
            };

            Controls.Add(button);
        }

        static void Main()
        {
            Application.Run(new Program());
        }
    }
}

WPF обеспечивает более гибкий и мощный подход к созданию приложений с GUI, используя декларативный язык разметки XAML (eXtensible Application Markup Language).

Пример создания простого приложения с использованием WPF:

<Window x:Class="MyWpfApp.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="MainWindow" Height="450" Width="800">
    <Grid>
        <Button Content="Click me" HorizontalAlignment="Center" VerticalAlignment="Center" Click="Button_Click"/>
    </Grid>
</Window>
            
using System.Windows;

namespace MyWpfApp
{
    public partial class MainWindow : Window
    {
        public MainWindow()
        {
            InitializeComponent();
        }

        private void Button_Click(object sender, RoutedEventArgs e)
        {
            MessageBox.Show("Button clicked!");
        }
    }
}
