Регулярные выражения

Регулярные выражения (Regex) используются для поиска и манипуляции строками. В C# для работы с регулярными выражениями используется пространство имен System.Text.RegularExpressions.


using System;
using System.Text.RegularExpressions;

class Program
{
    static void Main()
    {
        string input = "Hello, my phone number is 123-456-7890.";
        string pattern = @"\d{3}-\d{3}-\d{4}";
        Match match = Regex.Match(input, pattern);

        if (match.Success)
        {
            Console.WriteLine("Found phone number: " + match.Value);
        }
    }
}

Регулярные выражения позволяют находить шаблоны в строках и выполнять такие операции, как поиск, замена и разбиение строк.

Структуры. Перечисления

Структуры (struct) в C# - это пользовательские типы данных, которые позволяют объединить несколько переменных различных типов под одним именем.


public struct Point
{
    public int X { get; set; }
    public int Y { get; set; }

    public Point(int x, int y)
    {
        X = x;
        Y = y;
    }
}

Перечисления (enum) используются для определения набора именованных констант. Это полезно для создания переменных, которые могут принимать одно из нескольких фиксированных значений.


public enum DaysOfWeek
{
    Sunday,
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday
}

class Program
{
    static void Main()
    {
        DaysOfWeek today = DaysOfWeek.Wednesday;
        Console.WriteLine("Today is: " + today);
    }
}

Отношение между классами

Отношения между классами в C# могут быть разных типов: ассоциация, агрегация, композиция и наследование.

  • Ассоциация: простое отношение между двумя классами, где один класс использует методы и свойства другого.
    • 
public class Teacher
{
    public string Name { get; set; }
}

public class Course
{
    public string CourseName { get; set; }
    public Teacher Teacher { get; set; } // Ассоциация
}
  • Агрегация: слабая форма отношения "часть-целое", где часть может существовать независимо от целого.
    • 
public class Engine
{
    public void Start() { Console.WriteLine("Engine started."); }
}

public class Car
{
    public Engine Engine { get; set; } // Агрегация

    public Car(Engine engine)
    {
        Engine = engine;
    }
}
  • Композиция: сильная форма отношения "часть-целое", где части не могут существовать отдельно от целого.
    • 
public class Wheel
{
    public void Rotate() { Console.WriteLine("Wheel is rotating."); }
}

public class Bicycle
{
    private Wheel frontWheel = new Wheel(); // Композиция
    private Wheel backWheel = new Wheel();  // Композиция

    public void Ride()
    {
        frontWheel.Rotate();
        backWheel.Rotate();
        Console.WriteLine("Bicycle is riding.");
    }
}
  • Наследование: отношение, при котором один класс (наследник) наследует свойства и методы другого класса (родителя).
    • 
public class Animal
{
    public void Eat() { Console.WriteLine("Eating..."); }
}

public class Dog : Animal
{
    public void Bark() { Console.WriteLine("Barking..."); }
}
