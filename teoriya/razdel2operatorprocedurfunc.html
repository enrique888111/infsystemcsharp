Операторы

Операторы в C# используются для выполнения различных операций над данными и переменными. Основные категории операторов включают:

  • Арифметические операторы:

    Используются для выполнения основных математических операций.

    
int a = 5;
int b = 3;
int sum = a + b;    // сложение
int difference = a - b; // вычитание
int product = a * b; // умножение
int quotient = a / b; // деление
int remainder = a % b; // остаток от деления
  • Логические операторы:

    Используются для выполнения логических операций над булевыми значениями.

    
bool x = true;
bool y = false;
bool result1 = x && y; // логическое И
bool result2 = x || y; // логическое ИЛИ
bool result3 = !x;     // логическое НЕ
  • Операторы сравнения:

    Используются для сравнения значений.

    
int a = 5;
int b = 3;
bool isEqual = a == b;    // равно
bool isNotEqual = a != b; // не равно
bool isGreater = a > b;   // больше
bool isLess = a < b;      // меньше
bool isGreaterOrEqual = a >= b; // больше или равно
bool isLessOrEqual = a <= b;    // меньше или равно
  • Операторы присваивания:

    Используются для присваивания значений переменным.

    
int a = 5;
a += 3; // эквивалентно a = a + 3;
a -= 2; // эквивалентно a = a - 2;
a *= 4; // эквивалентно a = a * 4;
a /= 2; // эквивалентно a = a / 2;
a %= 3; // эквивалентно a = a % 3;

Процедуры и функции – методы класса. Встроенные методы

Методы в C# представляют собой блоки кода, которые выполняют определенные действия и могут быть вызваны из других частей программы. Методы могут принимать параметры и возвращать значения.


public class MathOperations
{
    // Метод, который не возвращает значение (процедура)
    public void PrintMessage()
    {
        Console.WriteLine("Hello from MathOperations!");
    }

    // Метод, который принимает параметры и возвращает значение (функция)
    public int Add(int a, int b)
    {
        return a + b;
    }
}

Встроенные методы — это методы, которые предоставляются стандартными библиотеками C# и .NET. Они включают методы для работы со строками, числами, массивами и другими типами данных.

  • Методы для работы со строками: 
    
string message = "Hello, World!";
int length = message.Length; // Получить длину строки
string upperMessage = message.ToUpper(); // Преобразовать строку в верхний регистр
string lowerMessage = message.ToLower(); // Преобразовать строку в нижний регистр
bool containsWorld = message.Contains("World"); // Проверить, содержит ли строка подстроку "World"
  • Методы для работы с массивами: 
    
int[] numbers = { 1, 2, 3, 4, 5 };
Array.Sort(numbers); // Отсортировать массив
int index = Array.IndexOf(numbers, 3); // Найти индекс элемента со значением 3
Array.Reverse(numbers); // Обратить порядок элементов в массиве

Методы являются важной частью программирования на C#, так как они позволяют структурировать код, делают его повторно используемым и легко поддерживаемым.
