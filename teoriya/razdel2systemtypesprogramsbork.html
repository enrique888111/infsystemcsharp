Раздел 2

Система типов

В C# существует две основных категории типов: значимые и ссылочные.

  • Значимые типы:

    Значимые типы хранят данные непосредственно в памяти. К ним относятся простые типы (int, float, bool), структуры (struct) и перечисления (enum). Значимые типы располагаются в стеке, что обеспечивает быстрый доступ к данным.

  • Ссылочные типы:

    Ссылочные типы хранят ссылку на данные, которые располагаются в куче. К ним относятся классы (class), массивы и интерфейсы. Ссылочные типы поддерживают наследование и полиморфизм.

Структура программы. Переменные, операции и выражения

Программа на C# состоит из одного или нескольких файлов, каждый из которых содержит определения типов и членов. Основной структурный элемент программы — это класс, в котором определяются методы и переменные.

public class Program
{
    public static void Main(string[] args)
    {
        // Объявление переменных
        int number = 10;
        double pi = 3.14;
        string message = "Hello, World!";

        // Операции и выражения
        int sum = number + 20;
        double area = pi * 5 * 5;
        string greeting = message + " How are you?";

        // Вывод на консоль
        Console.WriteLine(sum);
        Console.WriteLine(area);
        Console.WriteLine(greeting);
    }
}

Переменные используются для хранения данных, а операции и выражения позволяют манипулировать этими данными. Основные типы операций включают арифметические, логические, побитовые и операции сравнения.

Сборки. Атрибуты. Директивы. Пространства имен

В C# сборка представляет собой файл (обычно .dll или .exe), который содержит промежуточный язык (IL) и метаданные. Сборки являются единицами развертывания и управления версиями в .NET.

using System.Reflection;

// Атрибуты
[assembly: AssemblyTitle("My Application")]
[assembly: AssemblyDescription("Описание приложения")]
[assembly: AssemblyVersion("1.0.0.0")]

namespace MyApp
{
    class Program
    {
        static void Main(string[] args)
        {
            // Код программы
        }
    }
}

Атрибуты предоставляют способ добавления метаданных в код. Они могут быть применены к сборкам, типам, методам и другим элементам.

Директивы (например, using) упрощают работу с пространствами имен, предоставляя короткие пути к типам, определенным в других сборках.

using System;
using System.Collections.Generic;

namespace MyApp
{
    class Program
    {
        static void Main(string[] args)
        {
            List names = new List { "Alice", "Bob", "Charlie" };
            foreach (string name in names)
            {
                Console.WriteLine(name);
            }
        }
    }
}

Пространства имен организуют классы и другие типы в логические группы и предотвращают конфликты имен.
