Раздел 4

Работа с базами данных (ADO.NET)

ADO.NET (ActiveX Data Objects .NET) - это набор библиотек .NET Framework, предназначенных для работы с данными из различных источников данных, включая реляционные базы данных, XML-файлы, сервисы веб-служб и другие.

С ADO.NET можно выполнять следующие задачи:

  • Подключение к базе данных.
  • Выполнение запросов к базе данных.
  • Извлечение данных из базы данных и их обработка.
  • Обновление данных в базе данных.
  • Управление транзакциями.

Пример подключения к базе данных MS SQL Server и выполнения запроса SELECT:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
 string connectionString = "Data Source=myServerAddress;Initial Catalog=myDataBase;
 User Id=myUsername;Password=myPassword;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            connection.Open();
            
            string sql = "SELECT Id, Name FROM MyTable";
            
            using (SqlCommand command = new SqlCommand(sql, connection))
            {
                using (SqlDataReader reader = command.ExecuteReader())
                {
                    while (reader.Read())
                    {
                        Console.WriteLine($"ID: {reader["Id"]}, Name: {reader["Name"]}");
                    }
                }
            }
        }
    }
}

Этот пример подключается к базе данных MS SQL Server, выполняет запрос SELECT и выводит результаты на консоль.
