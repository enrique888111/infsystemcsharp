Интерфейсы

Интерфейсы в C# представляют собой контракты, которые определяют поведение объекта. Они определяют только сигнатуры методов, но не их реализацию.


interface IPlayable
{
    void Play();
}

interface IRecordable
{
    void Record();
}

class Player : IPlayable, IRecordable
{
    public void Play() { Console.WriteLine("Playing..."); }
    public void Record() { Console.WriteLine("Recording..."); }
}

class Program
{
    static void Main()
    {
        Player player = new Player();
        player.Play();
        player.Record();
    }
}

Классы могут реализовывать несколько интерфейсов, что позволяет им обладать различным поведением.

Обработка исключительных ситуаций

Исключения в C# используются для обработки ошибок и других исключительных ситуаций. Для обработки исключений используются блоки try-catch-finally.


try
{
    int x = 10;
    int y = 0;
    int result = x / y;
    Console.WriteLine("Result: " + result); // Division by zero exception
}
catch (DivideByZeroException ex)
{
    Console.WriteLine("Error: " + ex.Message);
}
finally
{
    Console.WriteLine("Finally block.");
}

Блок try используется для помещения кода, который может вызвать исключение. Блок catch используется для обработки исключения. Блок finally выполняется всегда, независимо от того, было ли исключение.
