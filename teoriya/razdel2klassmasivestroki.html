Раздел 2

Классы

Класс в C# представляет собой шаблон или чертеж для создания объектов. Класс определяет свойства и методы, которые объект будет иметь.


public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }

    public void Introduce()
    {
        Console.WriteLine($"Hi, my name is {Name} and I am {Age} years old.");
    }
}

Объект является экземпляром класса. Создание объекта класса выполняется с помощью ключевого слова new.


Person person = new Person();
person.Name = "Alice";
person.Age = 30;
person.Introduce();

Массивы. Класс Array

Массивы в C# представляют собой коллекции фиксированной длины одного типа данных. Они позволяют хранить несколько значений в одной переменной.


int[] numbers = new int[5];
numbers[0] = 1;
numbers[1] = 2;
numbers[2] = 3;
numbers[3] = 4;
numbers[4] = 5;

foreach (int number in numbers)
{
    Console.WriteLine(number);
}

Класс Array предоставляет различные методы и свойства для работы с массивами.


int[] numbers = { 5, 3, 8, 1, 2 };
Array.Sort(numbers); // Сортировка массива

foreach (int number in numbers)
{
    Console.WriteLine(number);
}

int index = Array.IndexOf(numbers, 3); // Поиск индекса элемента
Console.WriteLine("Index of 3: " + index);

Символы и строки постоянной длины. Классы String и StringBuilder

В C# символы представлены типом char, который хранит один Unicode символ. Строки представлены классом String и являются неизменяемыми.


char letter = 'A';
string greeting = "Hello, World!";

Console.WriteLine(letter);
Console.WriteLine(greeting);

Класс String предоставляет множество методов для работы со строками, такие как конкатенация, сравнение, получение подстрок и т.д.


string firstName = "John";
string lastName = "Doe";
string fullName = firstName + " " + lastName;

Console.WriteLine(fullName);
Console.WriteLine("Length: " + fullName.Length);
Console.WriteLine("Substring: " + fullName.Substring(0, 4));

Для создания и манипуляции изменяемыми строками используется класс StringBuilder. Он эффективнее для частых операций изменения строк.


StringBuilder sb = new StringBuilder("Hello");
sb.Append(", World!");

Console.WriteLine(sb.ToString());

sb.Replace("World", "C#");
Console.WriteLine(sb.ToString());
